# Overview

As a software developer, it is important to understand the process of writing, testing, and sharing software using version control systems. This project demonstrates a simple "Hello World" program that outputs a message to the console.

The software prints a greeting message along with a short introduction. Although simple, this program represents the first step in learning how to build and publish software projects using modern development tools.

The purpose of creating this software is to strengthen my understanding of programming basics and the workflow involved in creating, documenting, and publishing code using Git and GitHub.

# Development Environment

The tools used to develop this software include:

- Visual Studio Code
